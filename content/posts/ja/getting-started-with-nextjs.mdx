## はじめに

Next.jsは、Webアプリケーションの構築をシンプルかつ効率的にする強力なReactフレームワークです。

## 主な機能

- **App Router**: ファイルシステムベースのルーティング
- **サーバーコンポーネント**: サーバーサイドでのコンポーネントレンダリング
- **TypeScript**: ファーストクラスのTypeScriptサポート

## クイックスタート

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```
