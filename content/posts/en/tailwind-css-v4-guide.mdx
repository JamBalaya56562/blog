## What's New in v4

Tailwind CSS v4 brings significant improvements to the developer experience.

## Key Changes

1. **CSS-first configuration**: No more `tailwind.config.js`
2. **Lightning CSS**: Faster builds with Lightning CSS
3. **New `@theme` directive**: Define design tokens in CSS

## Example

```css
@import "tailwindcss";

@theme inline {
  --color-primary: #3b82f6;
}
```
