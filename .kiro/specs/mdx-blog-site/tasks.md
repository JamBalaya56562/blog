# 実装計画: MDXブログサイト

## 概要

Next.js 16 App RouterとMDXを使用したブログサイトの実装計画。コンテンツレベルのi18n（英語・日本語）、Strategy patternによるコンテンツローダー（ローカル/GitHub）、タグフィルタリング、画像プロキシ、SEOメタデータ生成を段階的に実装する。Tailwind CSS v4、Geistフォント、Biome、Bun、fast-checkによるプロパティベーステストを使用する。

## 備考

- `*`マーク付きのタスクはオプションであり、MVP実装時にはスキップ可能
- 各タスクは特定の要件を参照しており、トレーサビリティを確保している
- チェックポイントで段階的な検証を行う
- プロパティテストはfast-checkを使用し、`bun:test`で実行する
- E2Eテストが必要な場合はPlaywrightを使用する
